# Collect.jl

*Data manipulation facilities for Julia.*

## Package Abstract

Collect.jl supports [`Query`s](https://github.com/davidagold/StructuredQueries.jl/) against Julia [`DataFrame`s](https://github.com/JuliaStats/DataFrames.jl).
